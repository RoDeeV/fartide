# fartide
## A *NEW* environment using flowcharts to make coding easier and effective, while still being powerful.
Fart IDE uses manny libraries to work.
You can do __a lot__ so far, but its still a bit buggy
**Send suggestions by dming @aefarkalibastafinfein on Discord!**
New node example:

```function MyCoolNode() {
    this.addInput("In", 0);
    this.addOutput("Out", 0);
}
MyCoolNode.title = "Cool Node";
MyCoolNode.prototype.onExecute = function () {
    this.setOutputData(0, this.getInputData(0));
};```
